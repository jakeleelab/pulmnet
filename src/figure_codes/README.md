## Data availability and protected files

This repository contains code used to generate figures for the manuscript. Some processed input files required for complete figure reproduction are not included in the initial public release because they may contain patient-level clinical information and/or germline SNP-informed copy-number data.

FACETS-derived files, including `*_hisens.Rdata`, sample-level FACETS output files, and other intermediate files containing SNP-level copy-number information, are not publicly distributed.

Where possible, de-identified summary-level or derived data tables are provided. Additional data may be made available through controlled access, a data use agreement, or upon reasonable request, subject to institutional approval.

De-identified patient-level basic genomic data for all 205 patients have already been released through the AACR Project GENIE. A direct link to these data is provided on the main repository page.

Please note that some figure panels, particularly copy-number plots requiring sample-level FACETS output, may not be fully reproducible from the public repository alone.
